# Playwright E2E QA Assignment

This project contains an end-to-end Playwright automation suite for the Practice Test Automation login page.

## What is included
- Playwright tests for valid and invalid login flows
- Browser execution for Chromium and Firefox
- Screenshots captured at important steps
- QA documentation and a CSV test case catalog

## Run locally
```bash
npm install
npx playwright install chromium firefox
npx playwright test
```
