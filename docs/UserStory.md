# User Story

## Title
Login flow automation for Practice Test Automation

## Description
As a QA engineer, I want an automated end-to-end login workflow so that login behavior can be validated quickly and consistently across browsers.

## Acceptance Criteria
- The login page loads successfully.
- Valid credentials redirect the user to the authenticated page.
- Invalid credentials show an error state.
- Screenshots are captured for key steps.

## Business Value
This reduces manual regression effort and increases confidence in the authentication flow.
